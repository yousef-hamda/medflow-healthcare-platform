# Generated stubs (not committed)

Language stubs for `medflow/v1/events.proto` are generated in CI with
[buf](https://buf.build) and published as build artifacts — they are **not**
committed to the repository.

```bash
# from packages/proto/
buf lint
buf generate   # writes gen/ts, gen/python, gen/java per buf.gen.yaml
```

CI pins the buf CLI version via the `buf` setup action; plugin versions are
pinned in `buf.gen.yaml`. Everything under `gen/` except this README is
git-ignored.
